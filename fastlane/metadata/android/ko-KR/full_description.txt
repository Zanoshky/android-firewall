휴대폰의 인터넷 접근을 완전히 제어하세요. Firewall은 어떤 앱이 온라인에 접속할 수 있는지 결정하고, 기기 전체에서 트래커, 광고, 핑거프린팅을 차단합니다. 루팅 불필요. 광고 없음. 데이터 수집 없음. 무료 오픈 소스.

FIREWALL이 다른 점
기본적으로 모든 것을 차단합니다. 신뢰하는 앱만 허용합니다. 시스템 앱, 블로트웨어, 백그라운드 서비스는 조용히 차단됩니다. 데이터는 기기에 남습니다. 원격 서버 없음, 계정 없음.

작동 방식
기기에 로컬 VPN을 생성합니다. 모든 트래픽이 이를 통과합니다. 차단된 앱은 연결 불가. 허용된 앱은 정상 연결. VPN은 완전히 로컬입니다.

앱별 인터넷 제어
- 모든 앱이 기본적으로 차단
- 앱별 Wi-Fi 및 모바일 데이터 개별 토글
- 시스템 앱 및 블로트웨어 차단 (MIUI, Samsung, Huawei 등)
- 모두 허용 및 모두 차단 버튼
- 전체, 사용자, 시스템으로 필터링
- 재시작 없이 즉시 적용

트래커, 광고, 핑거프린트 차단
모든 DNS 쿼리를 가로채고 알려진 도메인 목록과 대조합니다. 일치하는 도메인은 NXDOMAIN으로 즉시 차단됩니다.

- 130개 이상의 도메인이 기본 차단
- 커뮤니티 목록: HaGeZi Light, OISD Small, 1Hosts Lite, Steven Black, AdGuard DNS
- 사용자 정의 차단 도메인 추가
- 오탐 방지 화이트리스트
- 방화벽과 독립적으로 토글

DNS OVER HTTPS (암호화 DNS)
ISP는 방문하는 모든 도메인을 볼 수 있습니다. DoH는 쿼리를 암호화합니다.

- Cloudflare, Google, Quad9 중 선택
- 한 번의 탭으로 제공업체 전환
- DoH 독립적으로 켜기/끄기
- 트래커 차단과 함께 작동

상시 보호
- 부팅 시 자동 시작 (잠금 해제 전에도)
- 앱 업데이트 후 자동 재시작
- 다이렉트 부트 지원
- 알림 셰이드의 빠른 설정 타일

실시간 연결 로그
- 모든 연결 시도의 실시간 로그
- 대상 IP, 포트, 프로토콜, 도메인 이름
- 색상 코드: 녹색, 빨간색, 주황색
- 앱 이름 또는 도메인으로 필터링

트래픽 통계
- 다운로드 및 업로드 합계
- 차단, 허용, 전체 연결 수
- 트래픽별 상위 앱

개인정보 보호 및 신뢰
- 데이터 수집 제로
- 광고 SDK나 서드파티 서비스 없음
- 오픈 소스 - 모든 코드 감사 가능
